## Commits by Label

- **Feature**
  > Adds a new feature to the app
- **Revision**
  > Major changes to the codes  and functions / methods
- **Fix**
  > Bugfixes
- **Optimization**
  > Minor improvements to codes
- **GUI**
  > Updates to UI and Form designs
- **Docs**
  > Updates to UTechnical documents and text files
